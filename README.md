# אתר עמנואל — הוראות שימוש

האתר מוכן וריק, עם 6 עמודים:

- `index.html` — עמוד הבית
- `history.html` — ראשית ההתיישבות
- `community.html` — חיי הקהילה
- `institutions.html` — מוסדות ובתי כנסת
- `gallery.html` — גלריית תמונות
- `links.html` — קישורים ומקורות

הנושאים לעמודים הם רק הצעה — אפשר לשנות שמות, למחוק עמוד, או להוסיף עמוד נוסף (ראו למטה "איך מוסיפים עמוד חדש").

## איך פותחים ועורכים את האתר

כל הקבצים הם HTML פשוט. אפשר:
1. לפתוח כל קובץ `.html` בדפדפן (לחיצה כפולה על הקובץ) כדי לראות איך הוא נראה.
2. לערוך את הקבצים בעורך טקסט כלשהו (למשל Notepad, TextEdit, או VS Code) כדי להוסיף תוכן.

## איך מוסיפים תמונה

בכל מקום באתר שיש תיבה עם מסגרת מקווקוות (`img-placeholder`) — זה מקום ריק שמחכה לתמונה. לדוגמה:

```html
<div class="img-placeholder">
  <svg ...>...</svg>
  <span class="ph-label">תמונה ראשית לעמוד</span>
  <span class="ph-file">images/history-hero.jpg</span>
</div>
```

כדי להוסיף תמונה במקום הזה:

1. שמרו את קובץ התמונה בתוך התיקייה `images` (היא כבר קיימת בתיקיית האתר), בשם המתאים — למשל `history-hero.jpg` (השם מופיע בשורה האחרונה של התיבה, `ph-file`).
2. מחקו את כל תיבת ה-`div` הזו (מ־`<div class="img-placeholder">` ועד ה־`</div>` הסוגר שלה).
3. הדביקו במקומה שורה כזו:

```html
<img src="images/history-hero.jpg" alt="תיאור קצר של התמונה">
```

וזהו — התמונה תופיע במקום התיבה הריקה.

## איך מחליפים טקסט

כל מקום שכתוב בו טקסט בסוגריים מרובעים, כמו:

```
[כתבו כאן על השנים הראשונות...]
```

זה טקסט לדוגמה בלבד. פשוט מחקו אותו וכתבו את הטקסט האמיתי במקומו, בתוך אותה תגית `<p>`. אפשר גם למחוק את ה־`class="text-placeholder"` מהתגית כדי שהטקסט האמיתי לא יופיע באפור-נטוי, אלא בעיצוב טקסט רגיל.

## איך מוסיפים תמונה נוספת לגלריה (`gallery.html`)

בתוך `gallery.html` יש רשימת תיבות `img-placeholder` בתוך `<div class="gallery-grid">`. כדי להוסיף עוד תמונה — העתיקו תיבה קיימת, הדביקו אותה בסוף הרשימה, ושנו את שם הקובץ בשורת `ph-file`. אחר כך החליפו את התיבה בתמונה כמו שמוסבר לעיל.

## איך מוסיפים קישור (`links.html`)

בכל קבוצת קישורים יש רשימת `<li>`. כדי להוסיף קישור חדש — העתיקו `<li>` קיים, הדביקו אותו בתוך אותה קבוצה (`<ul class="link-list">`), ושנו:
- את הכתובת ב־`href="#"` לכתובת האמיתית, למשל `href="https://example.com"`
- את השם והתיאור בטקסט

## איך מוסיפים עמוד (נושא) חדש

1. העתיקו אחד הקבצים הקיימים (למשל `community.html`) ותנו לעותק שם חדש, למשל `rabbis.html`.
2. שנו את הכותרת (`<title>` ו-`<h1>`) ואת התוכן בקובץ החדש.
3. הוסיפו קישור לעמוד החדש בתפריט הניווט — יש להוסיף שורה כמו:
   ```html
   <a href="rabbis.html">שם הנושא</a>
   ```
   בתוך תגית `<nav class="main-nav">` — **בכל אחד מקבצי ה-HTML הקיימים** (כדי שהתפריט יהיה זהה בכל העמודים).

## שינוי שם האתר / הכותרת העליונה

הכותרת "עמנואל — סיפור ההתיישבות החסידית" מופיעה בתוך `<p class="site-title">` בראש כל עמוד. כדי לשנות אותה — יש לערוך את השורה הזו **בכל אחד מקבצי ה-HTML**.

## העלאת האתר לאינטרנט

כשהתוכן מוכן, אפשר להעלות את כל התיקייה (כולל `images`) לכל שירות אחסון אתרים סטטיים (למשל GitHub Pages, Netlify, או שירות אחסון רגיל). אין צורך בשרת מיוחד — זה אתר סטטי פשוט.
